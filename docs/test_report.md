# Отчет о тестировании программы "Управление маршрутами"

## Информация о тестировании
- **Дата тестирования:** 28 февраля 2026 года
- **Версия программы:** 1.0
- **Тестировщик:** Нейфельд
- **Максимальное количество маршрутов:** 10
- **ОС для тестирования:** macOS

## Цель тестирования
Проверка работоспособности всех функций программы согласно заданию:
1. Создание массива записей о маршрутах (ввод с клавиатуры)
2. Определение маршрута с минимальной длиной
3. Сортировка записей по возрастанию длины маршрута
4. Поиск маршрутов по начальному или конечному пункту

## Результаты тестирования

### Тест 1: Добавление маршрутов (Функция создания массива)

**Цель:** Проверить возможность добавления до 10 маршрутов

**Процедура:**
1. Запустить программу
2. Выбрать пункт меню 1 (Add new marshrut)
3. Ввести данные для 5 маршрутов из тестового набора №1
4. Выбрать пункт меню 5 (Display all marshruts)

**Ожидаемый результат:** Все 5 маршрутов успешно добавлены и отображаются корректно

**Фактический результат:**
```
Num | Start | End | Length
------+----------------------+----------------------+----------
101 | Minsk | Brest | 350.00
102 | Gomel | Vitebsk | 300.00
103 | Grodno | Minsk | 270.00
104 | Mogilev | Gomel | 150.00
105 | Vitebsk | Polotsk | 100.00
```

**Статус:** ПРОЙДЕН
